一个典型的 C 程序内存特征将会有如下几个区域：

  • 文本段（Text segment）
  • 已初始化数据段（Initialized data segment）
  • 未初始化数据段（Uninitialized data segment）
  • 栈（Stack）
  • 堆（Heap）
